# **OpenHarmony 6.0.0 Release 测试报告**

## **1** **概述**

 本报告针对OpenHarmony 6.0.0 Release版本的测试验证及质量状态评估，包括对子系统特性和整机专项。

 测试版本： 

| **版本名称**                      | **测试起始时间** | **测试结束时间** |
| --------------------------------- | ---------------- | ---------------- |
| OpenHarmony 6.0.0.40~6.0.0.46版本 | 2025-07-07       | 2025-08-30       |

 测试环境：
